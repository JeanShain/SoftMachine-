Опис першого прототипу

# Назва проєкту
Програмне забезпечення побудови траеторії руху БПЛА з урахуванням зміни середовища
# Мета проєкту
Розробка програмної системи для моделювання руху безпілотного літального апарата (БПЛА) з можливістю побудови маршруту, візуалізації карти місцевості та анімації польоту.
# Основні можливості першого прототипу
На поточному етапі реалізовано базовий прототип системи, який включає:
- відображення карти місцевості;
- графічний інтерфейс користувача;
- візуалізацію БПЛА;
- побудову маршруту руху;
- анімацію переміщення дрона по маршруту;
- масштабування та навігацію по карті;
- завантаження графічних ресурсів;

# Використані технології
Проєкт реалізований з використанням:
мови програмування Python;
Створено на основі алгоритму А*;

# Архітектура проєкту
Проєкт поділений на декілька модулів:

- main.py -- Головний файл запуску програми.
- requirements.txt -- Список бібліотек, які треба встановити.
- config/settings.py -- Налаштування: розмір вікна, кількість клітинок, кольори, швидкість руху.
- core/node.py -- Опис однієї клітинки сітки.
- core/grid.py -- Створення карти/сітки, перевірка меж, сусідів, перешкод.
- core/astar.py -- Алгоритм A* для пошуку маршруту.
- core/environment.py -- Логіка зміни середовища: додавання або видалення перешкод, перебудова маршруту.
- core/dynamic_obstacle -- Створення динамічних перешкод.
- ui/pygame_visualizer.py -- Уся візуалізація через Pygame.
